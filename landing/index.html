▍왜 envyou 인가

🔒

Zero Cloud, Total Privacy

API 키·DB 패스워드는 외부 서버로 절대 전송되지 않습니다. AES-256-GCM으로 로컬에만 암호화 저장.

📌

Zero Context Switching

전역 단축키로 즉시 토글, 항상 위 고정(Always on Top). 에디터·터미널 이동 없이 바로 참조·복사.

🤖

MCP Native

Claude Desktop과 프로토콜로 연동. 대화형 코딩 중 환경 변수를 자동 주입·분석.

🖥️

80s Retro UI

Windows 1.0~3.1 감성의 픽셀 퍼펙트 베벨 UI. 가볍고(수 MB) 빠른 Tauri 기반.

▍AI와 안전하게 연동

envyou --mcp 모드로 Claude Desktop과 STDIO 프로토콜 통신. AI가 환경 변수를 읽거나 쓸 때 반드시 사용자 물리 승인 팝업이 떠야만 데이터가 전달됩니다.

claude_desktop_config.json
{
  "mcpServers": {
    "envyou": {
      "command": "/Applications/envyou.app/Contents/MacOS/envyou",
      "args": ["--mcp"],
      "env": {}
    }
  }
}

설정 메뉴의 “Link with Claude Desktop” 버튼 한 번이면 기존 MCP 서버를 보존한 채 자동 병합됩니다.
