왜 envyou 인가

무겁고 산만한 워크플로우에 지친 개발자를 위해, 딱 필요한 세 가지에 집중했습니다.

🪶

극한의 가벼움

Electron이 아닌 Tauri(Rust) 기반. 수 MB 메모리만 쓰는 플로팅 데스크톱 앱이 전역 단축키로 즉시 떠오릅니다.

⌨ Ctrl + Shift + E
🔒

완벽한 프라이버시

API 키·DB 패스워드는 어떤 서버로도 전송되지 않습니다. 단일 파일에 AES-256-GCM으로 암호화되어 오직 로컬에만 저장됩니다.
